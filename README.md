# Movie Ticket Booking System (React Project)

A modern Movie Ticket Booking System built using React.js, inspired by platforms like BookMyShow & Netflix-style UI. This project allows users to browse movies, view show timings, and understand booking flow through a clean, responsive interface.

---

## Features

* Display **Released & Upcoming Movies**
* Show movie show-times and dates
* Real-time ticket price concept
* Blog section for movie-related content
* Fully responsive UI (mobile & desktop)
* Built with React Functional Components
* Clean UI with CSS and Bootstrap 

---

## Tech Stack

* **Frontend:** React.js
* **Styling:** CSS & Bootstrap 
* **Version Control:** Git & GitHub

## How to Run the Project

1️ Clone the repository

```bash
git clone https://github.com/rahulpardeshi86/React_.git
```

2️ Go to project folder

```bash
cd React_
```

3️ Install dependencies

```bash
npm install
```

4️ Run the project

```bash
npm run dev
```
